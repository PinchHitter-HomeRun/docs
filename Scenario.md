# 시나리오

회원가입 ~ 대타 매칭까지의 시나리오입니다.

## 점장

1. 회원가입(지점 등록X)
2. 자신의 지점에 알바생 관리 (수락/거절 및 삭제)
3. 전체 대타 매칭된 경우 푸시 알람이 가며, 매칭에 대한 모든 정보 조회 가능

## 알바생(글쓴이)

1. 회원가입(지점 등록 필수X)(내 정보에서 추가 가능)
2. 지점 등록 안했을시 내 정보에서 지점 등록
3. 수락 후 대타가 필요한 경우 게시글 작성 가능
4. 게시글에 지점, 시간대, 시급, 주요 업무(+특이사항)를 작성하여 게시글 업로드.
5. 채팅요청 오면, 수락 / 거절
6. 채팅 후 매칭 수락 /거절

## 알바생(대타자)

1. 회원가입(지점 등록 필수X)(내 정보에서 추가 가능)
2. 지점 등록 안했을시 내 정보에서 지점 등록
3. 원하는 카테고리 선택 후 게시글 조회 가능
4. 게시글에 대하여 찜을 하거나 채팅요청 및 매칭요청
